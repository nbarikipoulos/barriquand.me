# www.barriquand.me

## Steps

### un/cipher data:

```js
  gulp uncipher -p 'my secret password' // from .cjson to .json
  gulp cipher -p 'my secret password' // from .json to .cjson
```

### Generate (production)

```js
  NODE_ENV=production gulp
```
